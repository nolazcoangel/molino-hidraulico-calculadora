# Calculadora de Molino Hidráulico

Aplicación web desarrollada en **Python** y **Streamlit** para el cálculo
del caudal en un molino hidráulico, utilizando métodos de Mecánica de Fluidos.

## Métodos implementados
- Método volumétrico
- Método analítico (Q = A · v)
- Comparación entre métodos

## Funcionalidades
- Ingreso interactivo de datos
- Cálculo automático del caudal
- Visualización clara de resultados
- Exportación de resultados a Excel

## Tecnologías utilizadas
- Python
- Streamlit
- Pandas
- Matemáticas aplicadas a Mecánica de Fluidos

Curso: Mecánica de Fluidos  
Tema: Molino hidráulico
